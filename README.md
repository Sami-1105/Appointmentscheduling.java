# Java Appointment Scheduling System

# Overview
A simple Java console application that allows users to register, schedule appointments, and view reports of their appointments.

# Features
- User registration and authentication
- Appointment scheduling with conflict detection
- Listing of user's appointments

# Technologies Used
- Java SE 8+
- VS Code
